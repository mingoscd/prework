Exercise 2:Imagine you are eleven years old, and in the five minutes before the end of the lesson, your Math teacher decides he should make his class more "fun" by introducing a "game". He explains that he is going to point at each pupil in turn and ask them to say the next number in sequence, starting from one. The "fun" part is that if the number is divisible by three, you instead say "Fizz" and if it is divisible by five you say "Buzz".

( Note: the online script set the number of students as 19 by default )


fizzbuzz.rb 
def fizzbuzz(n)
	if n%3==0 && n%5==0
		return "FizzBuzz"
	elsif n%3==0
		return "Fizz"
	elsif n%5==0
		return "Buzz"
	else
		return n
	end 		
end

puts "How many students are in the class?"
number = gets.chomp.to_i
array = [*1..number]

for i in array
	response = fizzbuzz(i)
	puts "#{response}"
end
